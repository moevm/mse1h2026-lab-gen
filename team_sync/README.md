## Шаблон для протокола
Протокол каждого созвона оформляется отдельным файлом в данной директории с названием `Созвон №X (dd.mm).md` 
```md
# Командный созвон №X
## Агенда
- **Дата**: dd.mm
- **Тема встречи**: Тема встречи
- **План встречи**:
  - Пункт 1
  - ...
- **Участники**:
  - ФИО участника №1
  - ...
- **Протоколист**: ФИО
- **Запись встречи**: url
 
## Протокол встречи
```
