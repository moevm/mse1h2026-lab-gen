## Установка и запуск
Для запуска нужно:

- скачать python3.10
- перейти в папку lab1_intro
- запустить ``` python3 main.py ```
- Далее программа сгенерирует тесты и спросит, есть ли в директории solution.py. Добавьте его и нажмите Y.
